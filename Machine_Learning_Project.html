Project Overview

In Fall 2024, I worked on a semester-long journey to use machine learning models for cancer subtyping. While this is already a widely researched field, my group’s analysis focused on comparing varying levels of data complexity and analyzing their effects on subtyping accuracy. For instance, gene analysis is often high-dimensional, leading us to investigate: does classification accuracy improve with more genes, or do you only need a small subset of genes that explain the most variance in the data?

This project compared Support Vector Machines (SVMs), Random Forests, and Boosting Algorithms to evaluate the accuracy of breast cancer tissue subtyping. Our dataset included five different cancerous tissue types and one normal tissue type. The SVM, trained on the full dataset, achieved an impressive 97% accuracy, showcasing stellar performance given the small test size.
